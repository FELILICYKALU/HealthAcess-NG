# Imo State Healthcare Dashboard

This is a Streamlit web app that visualizes healthcare access data in Imo State, Nigeria.

## Features

- Interactive dashboard showing healthcare facilities by LGA
- Risk levels based on access and density
- Built using Python, Streamlit, and Plotly

## Project Files

- `app.py`: Main Streamlit app
- `Imo_Healthcare_Data_With_Risk.xlsx`: Cleaned dataset used in the app
- `requirements.txt`: Python packages
- `README.md`: Project description

## How to Run

```bash
streamlit run app.py
```
